- JWT无状态认证机制
- BCrypt密码加密
- Spring Security权限控制

- 密码必须使用BCrypt加密存储
- JWT token有过期时间，需要处理刷新逻辑
- 用户名和邮箱需要唯一性验证
- 前端需要处理token过期的情况
- 数据库字段长度要合理设置
- 接口要做好参数验证和错误处理
